# Incident Timeline

This document describes the timeline of events observed during the simulated attack activities in the Elastic SIEM monitoring lab.

---

## Incident Summary

During monitoring of the Windows endpoint, Elastic SIEM detected suspicious PowerShell activity and an unauthorized user account creation event.

These activities triggered alerts generated by the SIEM detection rules.

---

## Timeline of Events

| Time | Event |
|-----|------|
| T0 | Windows endpoint running with Elastic Agent installed |
| T1 | PowerShell executed using `ExecutionPolicy Bypass` |
| T2 | Encoded PowerShell command executed |
| T3 | Detection rule triggered PowerShell alert |
| T4 | Unauthorized account creation simulated |
| T5 | Detection rule triggered account creation alert |
| T6 | SOC investigation performed in SIEM |

---

## Events Observed

### PowerShell Execution

Detected using query:
process.name:powershell.exe

This indicates PowerShell process execution on the endpoint.

---

### Unauthorized Account Creation

Detected using query:


event.code:4720


This indicates that a new user account was created on the system.

---

## Investigation

The SOC analyst reviewed SIEM logs and correlated the events with the simulated commands executed during the lab exercise.

The events were confirmed as part of a controlled attack simulation.

---

## Outcome

The SIEM monitoring system successfully detected the simulated activities and generated alerts
